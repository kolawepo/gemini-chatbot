Gemini Medical Student Chatbot
A conversational AI chatbot built using the Google Gemini API and Python, developed as part of a Computers and Ethics course at Alabama A&M University.
About
This chatbot is designed with the personality of a medical student — knowledgeable, curious, and passionate about health and science. It responds to questions with a medical lens while maintaining a conversational and approachable tone.
What I learned

How to integrate and query the Google Gemini API using Python
Prompt engineering techniques to shape AI personality and behavior
Ethical considerations around AI in healthcare and medicine
Version control and project management using GitHub

Technologies Used

Python
Google Gemini API
Jupyter Notebook

Setup

Clone the repository
Add your own Gemini API key
Run the notebook in Jupyter
